# SQL Business Analysis Project

This project analyzes an e-commerce dataset using SQL to extract business insights and understand customer purchasing behavior.

The goal is to demonstrate practical data analysis skills including data exploration, business metrics analysis, and customer segmentation.

---

## Dataset

The dataset contains three tables:

customers.csv → customer information  
orders.csv → transaction data  
products.csv → product catalog  

---

## Analysis Questions

The SQL analysis focuses on answering key business questions:

• What are the top-selling products?  
• Which customers generate the most revenue?  
• What are the monthly revenue trends?  
• Which product categories perform best?
